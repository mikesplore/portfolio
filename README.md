# Mike Splore Portfolio

Personal portfolio built with React and Vite.

## Overview

This site highlights:
- projects and GitHub work
- timeline activity (GitHub + writing)
- hackathons, events, and bucket-list goals
- contact and CV pages

The UI is intentionally minimal, fast, and mobile-friendly.

## Tech Stack

- React
- Vite
- React Router
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
